# SegmentPro: AI-Powered Customer Segmentation

SegmentPro is a full-stack web application designed for retail and mall management to intelligently categorize customers using machine learning. By analyzing customer demographics and spending behavior, the application generates actionable business insights to optimize marketing strategies and increase revenue.

## 🚀 Live Demo
**[Click here to view the live SegmentPro Dashboard](https://segmentpro.onrender.com)**

## 🧠 Machine Learning Architecture
The core of SegmentPro relies on the **K-Means Clustering** algorithm to process customer data. 
* Analyzes Annual Income (k$) and Spending Score (1-100).
* Automatically determines the optimal business profile for each customer (e.g., Target Customers, Careful Savers, Impulse Spenders).
* Generates real-time, color-coded scatter plots and demographic bar charts using Matplotlib.

## 💻 Tech Stack
* **Backend:** Python, Django, Pandas, Scikit-learn
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Reporting:** html2pdf.js (Automated PDF generation)
* **Deployment:** Gunicorn, WhiteNoise, Render

## ⚙️ Key Features
1. **Dynamic CSV Processing:** Users can upload raw customer data for instant analysis.
2. **Executive Dashboard:** Calculates high-level metrics like Total Customers, Average Income, and Average Spending Score.
3. **Strategic Recommendations:** Maps mathematical clusters to real-world business actions.
4. **Exportable Reports:** One-click generation of comprehensive PDF business reports for management.
